<idea>
  **Design Movement**: Institutional Fintech (Bloomberg/FactSet Modernized)
  **Core Principles**: Data density, high contrast, immediate readability, trust.
  **Color Philosophy**: Dark mode default. Deep charcoal backgrounds (#0f1115) with vibrant signal colors (neon green for bullish, hot red for bearish). Text is crisp white or cool gray.
  **Layout Paradigm**: Dashboard-first. Fixed sidebar for navigation, dense data tables with sticky headers. Split screen for list + detail view.
  **Signature Elements**: Monospace numbers for alignment. "Signal pills" with glowing borders. Sparklines in tables.
  **Interaction Philosophy**: Keyboard shortcuts for navigation. Instant row expansion. Hover states reveal detailed metrics.
  **Animation**: Minimal and fast (150ms). subtle pulses for new real-time data.
  **Typography System**: Inter for UI, JetBrains Mono for all financial data and tickers.
</idea>
